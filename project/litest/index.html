Litest

v.2.0.0

GitHub

monochrome.エコシステムの、第1歩。

生まれ変わったLitest。 その全ては、monochrome.エコシステムのために。 以前の「アプリケーション機能を内蔵する」考えから、完全に「ユーティリティ」ソフトウェアに生まれ変わりました。 また、それに伴いUIも大幅に変更。以前よりも大きく実用性が上がっています。

ソフトウェアの管理もこれだけで。

Litestには、革新的なアプリストアおよびソフトウェア管理機能が内蔵されています。それは、「ソフトウェア」タブおよび「ストア」タブから利用できます。
「ストア」タブからmonoStoreを利用することができます。また、monoStoreはWebベースであるため、ウェブブラウザからも操作が行えます。
また、「ソフトウェア」タブからは、Litestを通じてインストールしたソフトウェアのアップデートの確認と実行などが可能になっています。

Litestを試してみよう。

Litestは、macOS、Windows、Linux向けの各パッケージを、、と言いたいところですが、現在開発中です！
